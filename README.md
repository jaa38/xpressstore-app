# XpressStore

A scalable offline-first mobile commerce and storefront application built with React Native, Expo, and TypeScript.

## Stack

- Expo
- React Native
- TypeScript
- Zustand
- TanStack Query
- SQLite
- MMKV
- Supabase

## Features

- Storefront management
- Inventory tracking
- Offline support
- Order management
- Authentication
- Payments

## Getting Started

```bash
npm install
npx expo start
